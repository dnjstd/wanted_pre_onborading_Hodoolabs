## 역할

- [`김재호`]
  - Next.js를 사용한 프로젝트 구현
  - Figma를 참고한 메인레이아웃 구현
  - React-slick을 이용한 자동 무한슬라이드 구현

- [`원소연`]
  - Nest.js 초기 세팅
  - Figma 참고하여 메인, Nav,Footer 레이아웃 구현
  - react-reveal 라이브러리를 활용하여 Text FadeIn 애니메이션 구현
  - keyframes를 활용하여 첫 화면 Mouse Scroll Down 애니메이션 구현
